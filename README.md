# TP – Visualización de Algoritmos

## Integrantes

## Algoritmos implementados
- Bubble Sort (sort_bubble.py)
- Selection Sort (sort_selection.py)
- Insertion Sort (sort_insertion.py)

## Notas de implementación
- Cada algoritmo implementa la interfaz `init(vals)` y `step()` provista por el visualizador.
- `step()` hace un micro-paso y devuelve el dict requerido: `{"a","b","swap","done"}`.
- Se incluyeron contadores simples de comparaciones y swaps dentro de cada módulo para facilitar métricas (pueden leerse si se imprime el módulo).
